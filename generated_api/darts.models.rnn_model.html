Recurrent Neural Networks¶

class darts.models.rnn_model.RNNModel(model: Union[str, torch.nn.modules.module.Module] = 'RNN', output_length: int = 1, hidden_size: int = 25, n_rnn_layers: int = 1, hidden_fc_sizes: Optional[List] = None, dropout: float = 0.0, **kwargs)[source]¶

Bases: darts.models.torch_forecasting_model.TorchForecastingModel

Recurrent Neural Network Model (RNNs).

This class provides three variants of RNNs:

  • Vanilla RNN

  • LSTM

  • GRU

Parameters

  • model – Either a string specifying the RNN module type (“RNN”, “LSTM” or “GRU”), or a PyTorch module with the same specifications as darts.models.rnn_model.RNNModule.

  • output_length – Number of time steps to be output by the forecasting module.

  • hidden_size – Size for feature maps for each hidden RNN layer (\(h_n\)).

  • n_rnn_layers – Number of layers in the RNN module.

  • hidden_fc_sizes – Sizes of hidden layers connecting the last hidden layer of the RNN module to the output, if any.

  • dropout – Fraction of neurons afected by Dropout.
