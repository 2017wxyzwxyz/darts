Standard Regression model¶

class darts.models.standard_regression_model.StandardRegressionModel(train_n_points: int, model=LinearRegression(fit_intercept=False, n_jobs=- 1))[source]¶

Bases: darts.models.regression_model.RegressionModel

Simple wrapper for regression models implementing a fit() predict() functions models (e.g., scikit-learn regression).

Parameters

  • train_n_points – The number of most recent points from the training (features and target) time series that will be used to train the regression model. If this is None, or if the provided training series contain fewer points, the maximum possible number of time steps will be used for training.

  • model – A regression model that implements fit() and predict() methods. Default: sklearn.linear_model.LinearRegression(n_jobs=-1, fit_intercept=False)
