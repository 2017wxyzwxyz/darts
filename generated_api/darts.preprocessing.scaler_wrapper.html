Scaler wrapper¶

class darts.preprocessing.scaler_wrapper.ScalerWrapper(scaler=MinMaxScaler())[source]¶

Bases: object

Generic wrapper class for using transformers/scalers that implement fit(), transform() and inverse_transform() methods (typically from scikit-learn) on TimeSeries.

Parameters

scaler – The transformer/scaler to transform the data. It must provide the fit(), transform() and inverse_transform() methods. Default: sklearn.preprocessing.MinMaxScaler(feature_range=(0, 1)); this will scale all the values of a time series between 0 and 1.
