Backtesting simulation¶

u8timeseries.backtesting.forecasting_simulation.simulate_forecast_ar(series: u8timeseries.timeseries.TimeSeries, model: u8timeseries.models.autoregressive_model.AutoRegressiveModel, start: pandas._libs.tslibs.timestamps.Timestamp, fcast_horizon_n: int, trim_to_series: bool = True, verbose=False) → u8timeseries.timeseries.TimeSeries[source]¶

Provides an environment for forecasting future values of the TimeSeries ‘series`.

This function predicts the fcast_horizon_n values for the TimeSeries series starting from the date start according to the auto-regressive model model.

Parameters

  • series – The TimeSeries to forecast.

  • model – The AutoRegressiveModel to use.

  • start – The first time at which a prediction is produced for a future time.

  • fcast_horizon_n – The number of future values to predict.

  • trim_to_series – Whether the predicted series has the end trimmed to match the end of the main series or not.

  • verbose – Whether to print progress or not.

Returns

A TimeSeries containing the fore-casted values of series over the horizon with respect to the model model.
