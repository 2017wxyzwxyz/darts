u8timeseries.models.baselines module¶

class u8timeseries.models.baselines.KthValueAgoBaseline(k: int = 1)[source]¶

Bases: u8timeseries.models.autoregressive_model.AutoRegressiveModel

A baseline model that always predict value of k time steps ago.

More precisely, at last know time value t, the prediction for t + i is given by the value at time t - k + i.

Parameters

k – An integer, determines how far to fetch the prediction value.
