u8timeseries.models.regressive_model¶

class u8timeseries.models.regressive_model.RegressiveModel[source]¶

Bases: abc.ABC

This is a base class for various implementations of multi-variate models - models predicting time series from one or several time series. It also allows to do ensembling.
