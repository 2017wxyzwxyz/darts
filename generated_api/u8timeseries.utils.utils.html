u8timeseries.utils.utils module¶

u8timeseries.utils.utils.retain_period_common_to_all(series: List[u8timeseries.timeseries.TimeSeries]) → List[u8timeseries.timeseries.TimeSeries][source]¶

Trims all series in the provided list, if necessary, so that the return time series have the same time index (corresponding to largest duration common to all series).

Raises an error if no such time index exists. :param series: :return:
